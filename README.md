## About the Animation

The interactive animation walks through five stages of the argument:

1. **Traditional strategic groups** — the tech industry pre-2015, with clear group boundaries and mobility barriers (Caves & Porter, 1977)
2. **Microsoft in core groups** — Microsoft's historical position in Enterprise Software and OS/Devices
3. **Microsoft crosses barriers** — GitHub acquisition (2018), $13B OpenAI investment, Azure growth
4. **AI dissolves the boundaries** — Copilot, Azure AI Foundry, GitHub Copilot, and MAI-Thinking-1 connect every group through a shared AI agent layer
5. **New landscape** — Microsoft sits above the group structure as an integrated platform ecosystem; the theoretical gap Caves & Porter cannot explain

Use the **Prev / Next** buttons to step through stages, or **Play** to auto-advance.
